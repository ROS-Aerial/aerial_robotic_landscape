Autopilot Suites

There are several autopilot suites available for control-boards for aerial vehicles. This page mostly have an overview of all of those including some comparison factors in them as well.

Autopilot suite startup year latest version OS licence ROS support supported vehicles
Ardupilot (GitHub) 2009 4.5.7 (10/24) GPL 3.0 yes Copters, Fixed wings, VTOL
Betaflight (GitHub) 2015 4.5.1 (06/24) GPL 3.0 no Copters
crazyflie-firmware (GitHub) 2011 2024.10 GPL 3.0 yes* Quadcopters, Flapping wings
DJI autopilot (GitHub) 2006 3.6 (02/2018) closed yes Copters, VTOLS
Paparazzi (GitHub) 2003 6.4.0 (04/24) GPL 2.0 no Copters, Fixed wings, VTOL
PX4 (GitHub) 2009 1.15.1 (11/24) BSD 3-Clause yes Copters, Fixed wings, VTOL
ROSflight (GitHub) 2019 1.3.0 (03/20) BSD 3-Clause yes Copters, fixed wings

*community provided support
